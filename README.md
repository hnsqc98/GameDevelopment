## 第四课作业：A类-模型制作应用
### 文件说明
* obj模型文件（两套）
* fbx模型文件
* .mb的Maya模型文件：`shenqichuan_crocodile.mb`
* Unity工程文件：`unity_sqc_crocodile`
* 模型的LOD数据（三套）

* 建模：使用Maya自己建了一只小鳄鱼的模型
* 贴图：对模型进行了UV展开，并自己绘制了贴图，学号与姓名标记在了小鳄鱼尾巴上
* 资源生产：导出了`obj，fbx，mb`等文件，导出了材质贴图和法线贴图
* Unity相关：将Maya中的模型导入Unity，导出了三套LOD数据
